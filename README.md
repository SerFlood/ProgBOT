# ProgBOT

Инструкция к установке:

  1 - Скачайте Prog.rar,
  2 - Распакуйте архив в удобную для вас папку,
  3 - Откройте файл под названием PROGBOT.bat,
  4 - Можете начинать общаться с ProgBOT.
